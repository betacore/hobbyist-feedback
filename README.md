# Hobbyist Feedback

Willkommen beim offiziellen Community-Feedback-Projekt von **Hobbyist** 💚

Hier kannst du:

- 🐛 Fehler melden
- ✨ neue Features vorschlagen
- 💬 Feedback geben
- 💡 Ideen diskutieren
- 👍 Vorschläge anderer bewerten

---

# 🌍 Was ist Hobbyist?

Hobbyist verbindet Menschen über gemeinsame Interessen, Hobbys, Vereine, Gruppen, Events und Communities.

Beispiele:

- Fußball
- Basketball
- Lesen
- Gaming
- Musik
- Vereine
- lokale Communities
- Turniere
- Freizeitaktivitäten

Ziel ist es:

- Menschen zusammenzubringen
- Vereinsleben zu fördern
- Communities aufzubauen
- reale soziale Interaktion zu stärken

---

# 🚀 Wie kann ich Feedback geben?

## 🐛 Bug melden

Wenn etwas nicht funktioniert:

- Fehler
- Abstürze
- falsches Verhalten
- UI-Probleme
- Performanceprobleme

Gehe zu:

```txt
Issues → New Issue → Bug Report
```

---

## ✨ Feature vorschlagen

Wenn du:

- eine neue Idee hast
- eine Verbesserung möchtest
- einen neuen Usecase vorschlagen willst

Gehe zu:

```txt
Issues → New Issue → Feature Request
```

---

## 💬 Diskussionen & Ideen

Für:

- Brainstorming
- allgemeine Diskussionen
- Communityideen
- Feedback
- Showcase

Nutze:

```txt
Discussions
```

# 🧩 Wie erstellt man ein gutes Ticket?

## Gute Tickets enthalten

### ✅ Was ist passiert?

Beschreibe das Problem möglichst klar.

### ✅ Was hast du erwartet?

Wie sollte es eigentlich funktionieren?

### ✅ Wie kann man den Fehler reproduzieren?

Schritte:

1. Öffne ...
2. Klicke auf ...
3. Fehler erscheint ...

### ✅ Plattform

- Android
- iOS
- Web/Desktop

### ✅ Screenshots

Wenn möglich:

- Bilder
- Videos
- Logs
