# 🌍 Was ist Hobbyist?

Hobbyist verbindet Menschen über gemeinsame Interessen, Hobbys, Vereine, Gruppen, Events und Communities.

Beispiele:

- Fußball
- Basketball
- Lesen
- Gaming
- Musik
- Vereine
- lokale Communities
- Turniere
- Freizeitaktivitäten

Ziel ist es:

- Menschen zusammenzubringen
- Vereinsleben zu fördern
- Communities aufzubauen
- reale soziale Interaktion zu stärken

# 👍 Voting & Community Feedback

Du kannst bestehende Bugs oder Features unterstützen.

## So funktioniert Voting auf GitHub

Öffne ein Issue oder eine Discussion und nutze:

```txt
👍 Reactions
```

Je mehr 👍 ein Beitrag erhält:
- desto wichtiger ist er für die Community
- desto besser können wir priorisieren
- desto sichtbarer wird das Thema

👉 Bitte erst suchen bevor du ein neues Ticket erstellst.

Wenn bereits ein passendes Ticket existiert:
- 👍 reagieren
- kommentieren
- zusätzliche Infos ergänzen

statt ein Duplikat zu erstellen.

# 🧩 Wie erstellt man ein gutes Ticket?

## Gute Tickets enthalten

### ✅ Was ist passiert?

Beschreibe das Problem möglichst klar.

### ✅ Was hast du erwartet?

Wie sollte es eigentlich funktionieren?

### ✅ Wie kann man den Fehler reproduzieren?

Schritte:

1. Öffne ...
2. Klicke auf ...
3. Fehler erscheint ...

- Android

- Web/Desktop

### ✅ Screenshots

Wenn möglich:

- Bilder
- Videos
- Logs
